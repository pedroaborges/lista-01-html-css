# LISTA 01 - PROGRAMAÇÃO PRA INTERNET

Disciplina: Programação para Internet I
Aluno: Pedro Antonio

Primeira Lista de Programação pra Internet I

Objetivos: Reforçar os entendimentos sobre HTML e CSS.

Exercício 1:
Criei a estrutura básica de um site em HTML, com título, parágrafo e configurações corretas de idioma e charset.

Exercício 2:
Organizei o site usando tags semânticas como header, nav, main, section e footer, além de adicionar três links no menu.

Exercício 3:
Criei um arquivo CSS externo para mudar a cor da página e estilizar elementos usando seletor de tag, classe e id.

Exercício 4:
Fiz uma div no formato de card aplicando width, padding, border, margin e box-sizing para mostrar o Box Model.

Exercício 5:
Mantive o código organizado, bem indentado e separado corretamente entre HTML e CSS.
